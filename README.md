## Getting Started

1. **Clone the repository**
   ```bash
   git clone https://github.com/your-username/fakebook.git
   ```

2. **Install dependencies**
   ```bash
   cd fakebook
   flutter pub get
   ```

3. **Run the app**
   ```bash
   flutter run
   ```
